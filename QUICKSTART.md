## Two Ways to Run

### Option 1: Quick Start (Docker Compose)

```bash
# 1. Run setup to generate secrets
./setup.sh

# 2. Start everything with Docker
docker-compose up -d

# 3. Open browser
http://localhost:3000
```

That's it! MongoDB and the app will start together.

### Option 2: Development Mode

```bash
# 1. Run setup
./setup.sh

# 2. Start MongoDB
docker run -d -p 27017:27017 --name mongodb mongo:latest

# 3. Start dev server
npm run dev

# 4. Open browser
http://localhost:3000
```

## Usage Flow

1. **Sign Up** - Create an account at http://localhost:3000
2. **Create Project** - Click "New Project" on the dashboard
3. **Add Variables** - Select environment tab, click "Add Variable"
4. **Export** - Click "Export .env" or "Export JSON"
5. **View History** - Click "View History" to see audit log

## API Endpoints

See [API.md](./API.md) for complete API documentation.

**Authentication:**
- `POST /api/auth/signup` - Create account
- `POST /api/auth/login` - Login

**Projects:**
- `GET /api/projects` - List projects
- `POST /api/projects` - Create project
- `DELETE /api/projects/:id` - Delete project

**Variables:**
- `GET /api/projects/:id/variables?environment=dev` - List variables
- `POST /api/projects/:id/variables` - Create variable
- `PUT /api/projects/:id/variables/:variableId` - Update variable
- `DELETE /api/projects/:id/variables/:variableId` - Delete variable

**Export:**
- `GET /api/projects/:id/export?environment=dev&format=env` - Export .env
- `GET /api/projects/:id/export?environment=dev&format=json` - Export JSON

**Audit:**
- `GET /api/projects/:id/audit` - View change history

## Configuration

Edit `.env` file (created by setup.sh):

```bash
MONGODB_URI=mongodb://localhost:27017/envbox
JWT_SECRET=<auto-generated>
ENCRYPTION_KEY=<auto-generated>
NEXT_PUBLIC_APP_URL=http://localhost:3000
```

## Production Deployment

1. Update `.env` with production values
2. Use strong secrets (generated by setup.sh)
3. Enable HTTPS
4. Secure MongoDB with authentication
5. Run with Docker Compose:

```bash
docker-compose up -d
```
